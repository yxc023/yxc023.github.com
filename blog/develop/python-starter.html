虚拟环境 Virtualenv

python安装类库为全局安装, 在一台机器上一个用户安装一个类库, 对该机器上所有用户和项目有效. 所以为了解决这个问题, 引入了虚拟环境, 使依赖类库管理编程项目范围.

  • python 3.*版本自带
  • python 2.*须另外安装
